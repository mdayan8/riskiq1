# RiskIQ MVP

RiskIQ is an autonomous multi-agent financial compliance and decision intelligence MVP.

## Services
- Frontend: React + Vite (`/frontend`)
- API Gateway: Node.js + Express (`/backend-node`)
- AI Service: FastAPI + DeepSeek (`/ai-service-python`)
- Databases: PostgreSQL + MongoDB
